HTML for Short Quotations: The HTML tag defines a short quotation. Browsers normally insert quotation marks around the quotation.

Browsers usually insert quotation marks around the q element.

WWF's goal is to: Build a future where people live in harmony with nature.

HTML for Abbreviations: The HTML tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

The WHO was founded in 1948.

Marking up abbreviations can give useful information to browsers, translation systems and search-engines.

HTML for Work Title: The HTML tag defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.).

The HTML cite element defines the title of a work.

Browsers usually display cite elements in italic.
